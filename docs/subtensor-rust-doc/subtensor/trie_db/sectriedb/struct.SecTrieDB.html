pub struct SecTrieDB<'db, 'cache, L>
where

{ /* private fields */ }
Expand description

A Trie implementation which hashes keys and uses a generic HashDB backing database.

Use it as a Trie trait object. You can use raw() to get the backing TrieDB object.

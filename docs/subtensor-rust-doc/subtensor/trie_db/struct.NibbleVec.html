Struct NibbleVec

pub struct NibbleVec { /* private fields */ }
Expand description

Owning, nibble-oriented byte vector. Counterpart to NibbleSlice. Nibbles are always left aligned, so making a NibbleVec from a NibbleSlice can get costy.

    RFC822Name(&'a str),

    IPAddress(&'a [u8]),
    RegisteredID(Oid<'a>),
}
Expand description

Represents a GeneralName as defined in RFC5280. There is no support X.400 addresses and EDIPartyName.

String formats are not validated.

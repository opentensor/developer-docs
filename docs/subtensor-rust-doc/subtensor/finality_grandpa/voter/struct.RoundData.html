Fields§

§voter_id: Option<Id>

Local voter id (if any.)

§prevote_timer: Timer

Timer before prevotes can be cast. This should be Start + 2T where T is the gossip time estimate.

§precommit_timer: Timer

Timer before precommits can be cast. This should be Start + 4T

§incoming: Input

Incoming messages.

Outgoing messages.

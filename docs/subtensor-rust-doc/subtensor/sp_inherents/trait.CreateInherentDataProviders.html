Expand description

Something that can create inherent data providers.

It is possible for the caller to provide custom arguments to the callee by setting the ExtraArgs generic parameter.

The crate already provides some convince implementations of this trait for Box<dyn CreateInherentDataProviders> and closures. So, it should not be required to implement this trait manually.

pub fn run_in_context<R>(run: impl FnOnce() -> R) -> R
Expand description

Run the given closure run in a dispatch context.

Nested calls to this function will execute run in the same dispatch context as the initial call to this function. In other words, all nested calls of this function will be done in the same dispatch context.

Struct Behaviour

pub struct Behaviour { /* private fields */ }
Expand description

Network behaviour that automatically identifies nodes periodically, returns information about them, and answers identify queries from other nodes.

All external addresses of the local node supposedly observed by remotes are reported via ToSwarm::NewExternalAddrCandidate.

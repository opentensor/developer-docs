Block resource limits configuration structures.

FRAME defines two resources that are limited within a block:

  • Weight (execution cost/time)
  • Length (block size)

frame_system tracks consumption of each of these resources separately for each DispatchClass. This module contains configuration object for both resources, which should be passed to frame_system configuration when runtime is being set up.

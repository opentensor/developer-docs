fn import_notification_stream(&self) -> ImportNotifications<Block>

Get block import event stream.

Not guaranteed to be fired for every imported block, only fired when the node has synced to the tip or there is a re-org. Use every_import_notification_stream() if you want a notification of every imported block regardless.

litep2p::substream

Struct Substream

Source 
pub struct Substream { /* private fields */ }
Expand description

Litep2p substream type.

Implements tokio::io::AsyncRead/tokio::io::AsyncWrite traits which can be wrapped in a Framed to implement a custom codec.

In case a codec for the protocol was specified, Sink::send()/Stream::next() are also provided which implement the necessary framing to read/write codec-encoded messages from the underlying socket.

pub async fn send_framed(&mut self, bytes: Bytes) -> Result<()>

Send framed data to remote peer.

This function may be faster than the provided futures::Sink implementation for Substream as it has direct access to the API of the underlying socket as opposed to going through tokio::io::AsyncWrite.

ยงCancel safety

This method is not cancellation safe. If that is required, use the provided futures::Sink implementation.

Panics if no codec is provided.

#[task_index]
Expand description

Allows defining an index for a task.

This attribute is attached to a function inside an impl block annotated with pallet::tasks_experimental to define the index of a given work item.

It takes an integer literal as input, which is then used to define the index. This index should be unique for each function in the impl block.

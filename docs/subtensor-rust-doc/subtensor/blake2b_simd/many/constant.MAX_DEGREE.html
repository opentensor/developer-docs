The largest possible value of degree on the target platform.

Note that this constant reflects the parallelism degree supported by this crate, so it will change over time as support is added or removed. For example, when Rust stabilizes AVX-512 support and this crate adds an AVX-512 implementation, this constant will double on x86 targets.
